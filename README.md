# fahhh 
it plays the sound on every single keypress  
thats all it does lol  
ESC to exit
## Installation

1. Clone the repository:
   ```
   git clone https://github.com/Lim-JL/fahhh.git
   cd fahhh
   ```

2. (Optional) Create and activate a virtual environment:
   - Windows:
     ```
     python -m venv .venv
     .venv\Scripts\activate
     ```
   - Mac/Linux:
     ```
     python -m venv .venv
     source .venv/bin/activate
     ```

3. Install dependencies:
   ```
   pip install -r requirements.txt
   ```


4. Run the script:
   ```
   python main.py
   ```

5. shutdown:
	```
	esc key
	deactivate (to exit venv if applicable)
	```
